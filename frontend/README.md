# Frontend - React (Vite)

## Démarrage local

1. Installer les dépendances:
   - npm install
2. Lancer le serveur de dev:
   - npm run dev

## Pré-requis

Le backend FastAPI doit être lancé sur http://localhost:8000.

## Fonctionnalités MVP de ce squelette

- Inscription et connexion JWT
- Formulaire de création d'un bien
- Liste des biens publiés
- Page détail d'un bien
- Filtres avancés (ville, type, prix, surface, pièces)
- Demande client sur un bien + suivi des demandes selon rôle
- Dashboard KPI basique
- Estimation de prix via endpoint analytics
